API de Previsão do Tempo

Bem-vindo à documentação da nossa API de previsão do tempo. Esta API fornece informações meteorológicas para várias cidades. Abaixo estão descritos os endpoints disponíveis, os parâmetros retornados e exemplos de uso.

1. Lista de Cidades

Endpoint: /api/cidades

Método: GET

Descrição: Retorna a lista de todas as cidades disponíveis na API.

Exemplo de Resposta

{
    "cidades": [
        "Lisboa",
        "Porto",
        "Madrid",
        "Paris"
    ]
}

2. Previsão Meteorológica de Hoje

Endpoint: /api/previsao/hoje

Método: GET

Descrição: Retorna a previsão meteorológica para hoje para uma cidade específica.

Parâmetros:

  • cidade (obrigatório) - Nome da cidade para a qual deseja a previsão.

Exemplo de Resposta

{
    {
        "owner": "IPMA",
        "country": "PT",
        "data": [
        {
            "precipitaProb": "95.0",
            "tMin": "16.5",
            "tMax": "21.3",
            "predWindDir": "SW",
            "idWeatherType": 9,
            "classWindSpeed": 2,
            "longitude": "-9.1286",
            "forecastDate": "2024-06-08",
            "classPrecInt": 2,
            "latitude": "38.7660"
        },
        "globalIdLocal": 1110600,
        "dataUpdate": "2024-06-08T13:31:02"
    }
}
